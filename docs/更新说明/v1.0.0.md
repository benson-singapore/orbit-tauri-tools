# v1.0.0

**发布日期：** 2025 年 6 月  
**发布渠道：** 稳定版

Orbit 首个正式版本。整合插件市场、多形态阅读与本地 Runtime，可在同一应用中浏览文章、漫画、视频与图集。

---

## 本版本更新

### 插件与内容

- 插件市场：浏览、安装、更新官方插件
- 支持 RSS 自定义导入与 WASM / `.orbit` 插件包
- 插件分组管理：侧栏折叠、排序、自定义分组
- 体验模式：安全级 / 完整级，控制成人向内容展示

### 阅读与播放

- 六种浏览布局：阅读模式、瀑布流、卡片视图、阅览分屏、联播分屏、视频预览
- 文章阅读器：字体调节、代码高亮、图片代理加载
- 漫画：章节目录、连续翻页、页宽调节
- 视频：HLS 播放、YouTube 嵌入、视频墙同播
- 播放历史与断点续看（视频 / 音频 / 文章 / 漫画）
- 多窗口 Dock：同时挂起多篇文章或视频

### 其他

- 收藏与已读 / 未读标记
- Feed 搜索、分页加载、频道筛选
- AI 速读概括（需自行配置 LLM 提供商）
- 9 套主题色系与界面缩放
- 系统信息面板：应用 / Runtime 状态、数据库路径查看
