# v1.1.0

**发布日期：** 2026 年 6 月  
**发布渠道：** 稳定版

在 v1.0.0 基础上，本版本重点补齐社交内容形态、漫画阅读体验、应用内更新检查与多平台构建能力，并统一了主题样式与运行时网络请求。

---

## 本版本更新

### 插件与内容

- 插件管理器支持通过 URL 导入 `.orbit` 插件包，无需先下载到本地
- 新增社交插件类型：推文列表、短帖详情、社交视频播放
- 社交内容专属展示模式与宽度调节
- 预览模式按插件类型自动校验，避免为不支持的插件误选无效布局
- 社交插件频道栏显示控制与数据刷新、分页参数处理

### 阅读与播放

- 漫画文章改用专用页面组件渲染，替代原有 HTML 拼接方式
- 章节目录支持正序 / 倒序切换
- 改进漫画章节流的预加载与懒加载机制
- 文章内容图片支持点击预览与缩放查看
- YouTube 嵌入播放器增强，支持运行时基址与播放模式切换
- 优化图片代理绑定与加载失败处理

### 应用更新

- 系统信息面板新增「软件更新」页：检查更新、查看版本历史与发布说明
- 集成 Orbit 官方更新 API，自动识别当前平台并获取下载链接
- 规范化 macOS / Windows / Linux 平台标识显示

### 界面与主题

- 全面替换硬编码颜色为主题变量，按钮、边框、分割面板等组件随主题切换
- 插件市场卡片、评分视图、LLM 设置面板等视觉与交互优化
- 评分视图根据摘要内容动态调整行高与内边距
- 侧边栏底部插件商店按钮样式优化

### 运行时与构建

- 新增 `runtimeFetch`：桌面端本地请求走 Rust reqwest，外部请求走插件 HTTP，统一网络层
- macOS 增加 WASM JIT 运行时所需的安全权限配置
- 构建系统重构，支持 macOS（ARM64 / Intel）、Linux、Windows 交叉编译
- 新增 macOS 应用签名、公证与 DMG 打包脚本及配置模板
- 应用图标全面更新
